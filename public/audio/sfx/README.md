# SFX 音效檔案目錄

此目錄用於存放 UI 音效檔案。

## 所需音效清單

| 檔案名稱 | 用途 | 建議長度 |
|---------|------|---------|
| click.mp3 | 一般點擊 | 0.1-0.3s |
| hover.mp3 | 懸停 | 0.1s |
| select.mp3 | 選擇確認 | 0.2-0.4s |
| transition.mp3 | 場景轉換 | 0.5-1s |
| choice.mp3 | 選項選擇 | 0.2-0.4s |
| dialogue_advance.mp3 | 對話推進 | 0.1-0.2s |
| menu_open.mp3 | 選單開啟 | 0.2-0.3s |
| menu_close.mp3 | 選單關閉 | 0.2-0.3s |
| unlock.mp3 | 解鎖成就 | 0.5-1s |
| chapter_transition.mp3 | 章節過場（重要！） | 2-3s |

## 章節過場音效建議

`chapter_transition.mp3` 是最重要的音效，建議使用：
- 空靈的鐘聲或風鈴聲
- 緩慢的弦樂上升
- 神秘的合成器音效
- 時長約 2-3 秒，與視覺過場同步
