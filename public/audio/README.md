# 音訊資源資料夾

請將音訊檔案放置在對應的資料夾中：

## 資料夾結構

```
audio/
├── bgm/           # 背景音樂
│   ├── title.mp3           # 標題畫面
│   ├── preface.mp3         # 作者序
│   ├── prologue.mp3        # 序章
│   ├── chapter_calm.mp3    # 平靜章節
│   ├── chapter_tension.mp3 # 緊張章節
│   └── chapter_emotional.mp3 # 感性章節
│
├── sfx/           # 音效
│   ├── click.mp3           # 點擊
│   ├── hover.mp3           # 懸停
│   ├── select.mp3          # 選擇
│   ├── transition.mp3      # 轉場
│   ├── choice.mp3          # 做選擇
│   ├── dialogue_advance.mp3 # 對話推進
│   ├── menu_open.mp3       # 開啟選單
│   ├── menu_close.mp3      # 關閉選單
│   └── unlock.mp3          # 解鎖
│
└── ambient/       # 環境音效
    ├── void.mp3            # 虛空
    ├── ferry.mp3           # 渡口
    ├── training_ground.mp3 # 訓練場
    └── cosmos.mp3          # 宇宙
```

## 建議格式

- **格式**: MP3 (相容性最佳) 或 OGG (較小檔案)
- **背景音樂**: 建議 128-192 kbps
- **音效**: 建議 96-128 kbps
- **環境音效**: 建議製作可無縫循環的版本

## 音量建議

音訊系統會自動處理音量，但建議：
- 所有音訊正規化至 -3dB 左右
- 背景音樂和環境音效需可無縫循環
