# 音訊資源說明

## 動態 BGM 系統

本遊戲實作了動態 BGM 切換系統，根據劇情氛圍自動變換背景音樂：

| 氛圍類型 | 對應 BGM | 適用場景 |
|---------|---------|---------|
| calm（平靜） | chapter_calm.mp3 | 日常敘述、冥想、哲學思考 |
| tension（緊張） | chapter_tension.mp3 | 壓力場景、衝突、危機時刻、伊的對話 |
| emotional（情緒） | chapter_emotional.mp3 | 感性時刻、回憶、情感表達 |
| revelation（啟示） | revelation.mp3 | 頓悟、真相揭示、重要轉折 |
| mysterious（神秘） | mysterious.mp3 | 超自然元素、靈魂相關場景 |

## 資料夾結構

```
audio/
├── bgm/           # 背景音樂
│   ├── title.mp3           # 標題畫面

│   ├── chapter_calm.mp3    # 平靜氛圍
│   ├── chapter_tension.mp3 # 緊張氛圍
│   ├── chapter_emotional.mp3 # 感性氛圍
│   ├── revelation.mp3      # 啟示/頓悟時刻
│   └── mysterious.mp3      # 神秘氛圍
│
├── sfx/           # 音效
│   ├── click.mp3           # 點擊
│   ├── hover.mp3           # 懸停
│   ├── select.mp3          # 選擇
│   ├── transition.mp3      # 轉場
│   ├── choice.mp3          # 做選擇
│   ├── dialogue_advance.mp3 # 對話推進
│   ├── menu_open.mp3       # 開啟選單
│   ├── menu_close.mp3      # 關閉選單
│   └── unlock.mp3          # 解鎖
│
└── ambient/       # 環境音效
    ├── void.mp3            # 虛空（作者序、靈魂抽離場景）
    ├── ferry.mp3           # 渡口（迷霧渡口、問心初登場）
    ├── training_ground.mp3 # 訓練場（序章前半）
    ├── cosmos.mp3          # 宇宙（元壹境相關）
    ├── night.mp3           # 夜晚（刪除之夜、刪除鍵特寫）
    ├── meditation.mp3      # 冥想（王陽明、心即理場景）
    └── water.mp3           # 流水（擺渡人場景）
```

## 章節氛圍對照

### 序章（prologue）
- 1-25: 平靜哲學敘述
- 26-37: 神秘（提到靈魂、伊）
- 38-41: 緊張（伊登場）
- 42-57: 情緒（自我接納）

- 15-39: 情緒（回憶過去）
- 40-69: 緊張（問心對話）
- 70-99: 情緒（情感釋放）
- 100+: 啟示（頓悟時刻）

### 第二章 渡口
- 1-29: 神秘（渡口場景）
- 30-59: 平靜（對話理解）
- 60+: 啟示（領悟）

## 建議格式

- **格式**: MP3 (相容性最佳) 或 OGG (較小檔案)
- **背景音樂**: 建議 128-192 kbps
- **音效**: 建議 96-128 kbps
- **環境音效**: 建議製作可無縫循環的版本

## 音量建議

音訊系統會自動處理音量，但建議：
- 所有音訊正規化至 -3dB 左右
- 背景音樂和環境音效需可無縫循環
- BGM 會根據氛圍自動淡入淡出切換
