# 音訊資源資料夾

請將音訊檔案放置在對應的資料夾中：

## 資料夾結構

```
audio/
├── bgm/           # 背景音樂
│   ├── title.mp3           # 標題畫面
│   ├── preface.mp3         # 作者序
│   ├── prologue.mp3        # 序章
│   ├── chapter_calm.mp3    # 平靜章節
│   ├── chapter_tension.mp3 # 緊張章節
│   └── chapter_emotional.mp3 # 感性章節
│
├── sfx/           # 音效
│   ├── click.mp3           # 點擊
│   ├── hover.mp3           # 懸停
│   ├── select.mp3          # 選擇
│   ├── transition.mp3      # 轉場
│   ├── choice.mp3          # 做選擇
│   ├── dialogue_advance.mp3 # 對話推進
│   ├── menu_open.mp3       # 開啟選單
│   ├── menu_close.mp3      # 關閉選單
│   └── unlock.mp3          # 解鎖
│
└── ambient/       # 環境音效
    ├── void.mp3            # 虛空（作者序、靈魂抽離場景）
    ├── ferry.mp3           # 渡口（迷霧渡口、問心初登場）
    ├── training_ground.mp3 # 訓練場（序章前半）
    ├── cosmos.mp3          # 宇宙（元壹境相關）
    ├── night.mp3           # 夜晚（刪除之夜、刪除鍵特寫）
    ├── meditation.mp3      # 冥想（王陽明、心即理場景）
    └── water.mp3           # 流水（擺渡人場景）
```

## 場景與環境音對應

| 場景圖片 | 環境音效 | 建議音效內容 |
|---------|---------|-------------|
| 作者序 | void.mp3 | 空靈的虛空氛圍、微弱的風聲 |
| 訓練場 | training_ground.mp3 | 宇宙教室的低頻嗡鳴 |
| 問心初登場 | ferry.mp3 | 霧氣飄動、遠處水聲 |
| 刪除之夜 | night.mp3 | 寂靜的夜晚、偶爾的蟲鳴 |
| 刪除鍵特寫 | night.mp3 | 鍵盤敲擊聲、電腦風扇低鳴 |
| 靈魂抽離 | void.mp3 | 靈魂飄離的空靈感 |
| 迷霧渡口 | ferry.mp3 | 濃霧中的水波聲 |
| 擺渡人 | water.mp3 | 划船的水聲、槳聲 |
| 王陽明 | meditation.mp3 | 禪意的空靈音效、偶爾的木魚聲 |
| 心即理 | meditation.mp3 | 內心的覺醒、清澈的鐘聲 |

## 建議格式

- **格式**: MP3 (相容性最佳) 或 OGG (較小檔案)
- **背景音樂**: 建議 128-192 kbps
- **音效**: 建議 96-128 kbps
- **環境音效**: 建議製作可無縫循環的版本

## 音量建議

音訊系統會自動處理音量，但建議：
- 所有音訊正規化至 -3dB 左右
- 背景音樂和環境音效需可無縫循環
